Create a premium ecommerce storefront prototype.

Technology Stack:
- React
- Vite
- TypeScript
- Vanilla CSS (NO Tailwind CSS)
- No UI component libraries
- No CSS frameworks
- Traditional CSS architecture

Important Development Rules:
- Use CSS Variables extensively for colors, spacing, typography, border radius, shadows, transitions and layout measurements.
- Store all design tokens inside a global variables file.
- Follow reusable and maintainable CSS practices.
- Use semantic HTML.
- Write clean, scalable code.
- Create reusable React components.
- Keep styling modular and organized.
-Never hardcode colors directly inside CSS. Every color, spacing value, font size, shadow, border radius and transition duration must come from CSS variables

Design Direction:
- Light Theme Only
- Do NOT create dark mode support
- Premium, elegant and modern design
- Luxury brand appearance
- Clean whitespace
- Professional typography
- Subtle animations and hover effects
- Mobile-first responsive design
- High-end ecommerce feel
